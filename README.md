# WorldSim
## this is a world sim for AI model experimentation with a virtual world and VR technology integration
## the sim will use a new kind of engine.
## This will be ran initially on lower gfx settings on an RPi linux based hardware.

* Installation Instructions

1. clone the repo
2. open the project folder where you cloned the repo
3. create a cmakeproject 
4. move the files you cloned from the repo into the project folder.

5. open the project

6. this will come with at least a .exe file.
